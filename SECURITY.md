### Security Policy

Thank you for helping keep this project and its users safe.

## Reporting a Vulnerability

- Please report vulnerabilities responsibly. Do not create a public issue for security reports.
- Preferred methods:
  - Open a private security advisory in the repository (if enabled), or
  - Email the maintainers with details and a proof of concept.

Please include:

- Affected versions/areas (e.g., `src/lib/providers/openai.ts`, `src/app/api/**`)
- Steps to reproduce or proof of concept
- Expected vs. actual behavior
- Potential impact and suggested mitigations

We will acknowledge receipt within 72 hours and aim to provide an initial assessment within 7 days.

## Supported Versions

- The latest `main` branch and the most recent release are supported for security fixes.

## Disclosure

- We follow responsible disclosure. We will work with you to validate and fix the issue before public disclosure.
